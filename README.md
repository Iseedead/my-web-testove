# Testove
## Simple Vue.js/Silex/MongoDB app.

### Requisites
- php5.6
- composer
- node
- npm
- mongoDB

### Run
Install dependencies
```
$ composer install
$ npm install
```
Start
```
$ sudo systemctl start mongodb (or $ sudo service mongod start)
$ npm run start
```
Server will run on http://localhost:7001/
